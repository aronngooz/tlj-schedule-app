# 🥐Tours Les Jours Schedule App🥖

I am currently developing a mobile scheduling app for Tours Les Jours employees and managers. The purpose of this app is to make scheduling easier by giving users one place to log in, view schedules, and manage work-related scheduling tasks.

## Project Overview

I started this project to create a scheduling app similar to workforce scheduling platforms like MyTime, but customized for Tours Les Jours. My goal is to build an app that helps both employees and managers handle scheduling in a simple and organized way.

Employees will be able to:
- log in
- view their schedule
- see upcoming shifts

Managers will later be able to:
- create schedules
- assign shifts
- edit shifts
- manage employee scheduling information

## Current Features

Right now, my app includes:
- a login screen
- a home screen
- a schedule screen
- a basic mobile UI built with React Native and Expo

## Planned Features

As I continue developing the app, I plan to add:
- employee and manager role-based access
- shift creation and editing
- weekly calendar view
- time-off requests
- availability management
- shift swap requests
- notifications
- backend and database integration
